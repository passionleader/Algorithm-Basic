# 학습계획(예정)
## 전체 공부 순서
1. 개념 복습
2. 코딩 및 수식 정리
3. 블로그에 게시

## 공부할 목록
1. 알고리즘
2. 운영체제
3. 컴퓨터구조
4. 네트워크

### 알고리즘 공부 순서
1. 기본 자료구조
   1. 기본 정렬(insertion selection bubble merge)
   2. merge sort 및 재귀 점화식
   3. BST
      * AVL 및 AVL Sort
   4. Hash, Chaining
   5. 점화식 수학식같이 작성하는 방법
   6. Queue 와 Stack 구현
   7. 우선 순위 큐
   8. 힙, 힙정렬
   9. 해시 테이블
   10. 기수정렬
2. 그래프
   1. 그래프 알고리즘(무향 그래프, 유향 그래프)
   2. DFS BFS 구현
   3. 최소 신장 트리, 프림 알고리즘
   4. 크루스칼 알고리즘
      * 위상정렬
   5. 재귀/동적 알고리즘(피보나치 수 구해보기)
   6. 다익스트라, 그리디, 벨만 포드 
   7. P, NP,다항식 시간 변환
